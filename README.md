# APO – Fundamentos de Programação para Internet

Este projeto foi desenvolvido como parte da disciplina Algoritmos e Fundamentos de Programação (APO) da UNIPAR. O objetivo é criar um sistema simples em PHP capaz de capturar dados inseridos pelo usuário e gerar automaticamente uma página formatada com essas informações.

---

## Estrutura do Projeto

```
APO_Fundamentos_de_Programacao_para_Internet/
│
├── assets/           # Arquivos estáticos (CSS, JS, imagens)
├── views/            # Templates e páginas HTML
├── gerar.php         # Script que processa e exibe os dados enviados
└── index.php         # Página inicial
```

---

## Objetivo

O projeto consiste em:

* Capturar dados por meio de um formulário HTML
* Processar as informações enviadas através de PHP (método POST)
* Exibir uma página final com as informações organizadas de forma clara

O foco é reforçar conceitos básicos de lógica, formulários, variáveis e organização de arquivos.

---

## Como Executar o Projeto

### 1. Verifique se o PHP está instalado

Você pode usar XAMPP, WAMP ou PHP standalone.

### 2. Clone o repositório

```bash
git clone https://github.com/JDeLima35/APO_Fundamentos_de_Programacao_para_Internet.git
```

### 3. Acesse o diretório do projeto

```bash
cd APO_Fundamentos_de_Programacao_para_Internet
```

### 4. Inicie um servidor PHP local

```bash
php -S localhost:8000
```

### 5. Acesse pelo navegador

```
http://localhost:8000
```
